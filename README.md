# Mock API for time-tracker

External API imitator for https://github.com/nquidox/time-tracker/

## Setup
Edit .env file params if needed.
```sh
#Server address
FAKEAPI_HOST=127.0.0.1
FAKEAPI_PORT=9001

#SQLite DB
DB_NAME=database.db

#Loggers
API_LOG_LVL=Info
DB_LOG_LVL=Silent
```
